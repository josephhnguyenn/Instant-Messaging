# Instant-Messaging
Python Codes for Client and Server. 
Multi-threaded implementation for concurrently server request handling. 
Allows for multiple clients to be connected to the same network.

To Use: 
Server host must port forward their network to local IP: 192.168.1.152 with PORT 5006.
Users must have the host's public IP address entered within the Source Code
First, the Server host must execute the Server.py code. 
Then Users must execute the Client.py code, after entering in the Host's Public IP address and correct PORT.
